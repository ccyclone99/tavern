## 关键叙事记录

开局：玩家作为地表流浪者进入第7区避难所，面对老王、苏珊、阿杰和两百名幸存者的警惕。

信任阶段：玩家提交路线和辐射读数，接受体检并只公开无传染性结论。第三次行动以部分成功完成授权，说明委员会仍有怀疑，但允许一次受控探索。

路线阶段：玩家在旧商场辨认 B-17 通道，并通过旧终端取得完整路线日志。一次大成功使路线挑战提前完成，并自然切入新伊甸核验。

核验阶段：玩家确认空气循环可用，净水系统可修但存在阀门问题，随后读取容量与入口相关记录。新伊甸从“可能有补给”升级为“可迁徙的新家园”。

表决阶段：玩家公开证据链，随后说服老王承担政治责任，并用苏珊、阿杰的专业背书压住质疑。迁徙方案通过。

迁徙阶段：第一批工程队按安全路线抵达新伊甸，生命维持系统启动。末段以部分成功收束：系统亮起，但居民恐惧仍存在，适合作为后续支线或长期经营压力。

## 合理性判断

整体流程比旧版本合理：玩家不能靠一句“说服委员会”直接跳过主线，必须先建立最低信任，再拿路线证据，再验证新家园，最后表决和执行迁徙。NPC 信息也没有开局全公开，阿杰坐标、新伊甸可信度、委员会投票条件都通过阶段推进逐渐落地。

检定密度合适：12 次检定刚好落在 SPEC 的标准目标区间上限。流程里有 2 次部分成功、1 次大成功，既有推进感，也保留代价。尤其第三回合和第十二回合的部分成功，让“怀疑仍在”和“新家园仍需治理”有了后续空间。

本次测试暴露并已修复两个关键问题：

- 证据添加会触发场景归一化，旧挑战对象被替换，导致已完成挑战重新保持 active。已在 `resolveChallengeCheck` 中重新定位 live challenge 后再结算状态。
- 主线目标曾可能被单条证据提前放行，绕过未完成挑战。已增加“主线目标存在未完成绑定挑战时，证据/结论不能绕过挑战”的拦截。

后续可优化点：

- 模型偶尔返回空叙事时，当前 UI 会隐藏空白气泡；后续可以在后台记录空响应率，用于判断是否需要模型重试。
- 复合行动现在会记录同一挑战内的次级命中方法，并在大成功时给出一个额外证据收益；后续可以继续扩展为“主检定 + 次级代价/收益”的完整结算。
- 通关记录已增加按阶段回顾；后续可以把阶段回顾做成可展开时间线，展示每阶段的关键对话原文。
