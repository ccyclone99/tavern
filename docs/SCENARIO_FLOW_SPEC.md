# Tavern Scenario Flow SPEC

本文档定义下一阶段“剧本与游戏流程强化”的规格。目标是让每个预设世界不只是有设定和 NPC，而是有清晰的开局推进、可玩的中段循环、失败后的剧情转向，以及玩家卡住时的下一步提示。

## 1. 目标

### 1.1 每个剧本有可执行流程

每个预设世界必须至少包含：

- 核心冲突：玩家为什么必须行动。
- 开局行动：玩家前 10 分钟自然能做的 3-5 件事。
- 中段循环：调查、交涉、准备、行动、后果如何循环。
- 失败推进：失败不是停住，而是带来代价、线索或新局势。
- 卡住提示：玩家不知道做什么时，系统能给出 2-4 个自然语言建议。

### 1.2 流程引导不替玩家做决定

系统提供的是“可推动方向”，不是强制选择。建议必须是玩家可以直接输入或改写的句子，例如：

- `向审判官塞拉斯解释污染星球上的经历`
- `询问艾拉关于第三道影子的梦`
- `申请前往下层货舱检查异形遗物`

禁止把建议写成纯系统指令，例如：

- `推进主线`
- `触发第一节拍`
- `完成任务目标1`

### 1.3 私密信息仍需解锁

流程指南可以知道 NPC 秘密和剧情真相，但展示给玩家的建议只能指向“调查/询问/验证”，不能直接泄露答案。

正确：

- `试探克拉克斯为什么私自复制过遗物数据`

如果玩家尚未获得相关线索，则应更保守：

- `询问克拉克斯最近的引擎异常`

## 2. 数据结构

新增 `scene.flowGuide`：

```js
scene.flowGuide = {
  openingMoves: [],
  sessionGoals: [],
  stalledPrompts: [],
  failForward: [],
  completedMoves: []
}
```

字段含义：

- `openingMoves`：开局 3-5 个自然语言行动建议。
- `sessionGoals`：本剧本本次游玩应追踪的阶段目标。
- `stalledPrompts`：玩家卡住或输入“我该做什么”时可给出的建议。
- `failForward`：失败、部分成功、拖延时可产生的推进型后果。
- `completedMoves`：已由系统或后续逻辑标记完成的流程建议，当前阶段可先保留为空。

## 3. 推荐行动规则

`WorldEngine.getCurrentSituation()` 应结合以下来源构建 `recommendedActions`：

1. `flowGuide.openingMoves`。
2. 当前主线任务未完成目标。
3. 当前剧情弧节拍条件。
4. 紧急局势时钟。
5. 可见 NPC 反制。
6. 最近线索。
7. `flowGuide.stalledPrompts` 兜底。

排序原则：

- 开局前 3 回合优先展示 `openingMoves`。
- 中后段优先展示任务、时钟、反制、线索。
- 任意状态最多展示 4 条。
- 展示文案必须是玩家可直接输入的行动句。

## 4. Prompt 注入规则

PromptBuilder 应注入“剧本流程指南”块，提醒 DM：

- 用流程指南保持节奏。
- 不替玩家选择。
- 当玩家卡住时，用 NPC、环境或局势给出下一步。
- 失败时参考 `failForward` 生成新代价或新线索。
- 私密真相只能通过线索、关系、检定或剧情事件逐步解锁。

## 5. 预设世界要求

### 5.1 审判庭黑船

开局应围绕“嫌疑人如何获得最低限度信任”展开：

- 向塞拉斯解释污染星球经历。
- 让克拉克斯检查血样或设备记录。
- 接触艾拉，验证第三道影子的梦。
- 申请调查下层货舱遗物。

失败推进：

- 审判庭怀疑上升，但给出更明确的审讯问题。
- 船上时钟推进，货舱遗物低语增强。
- 艾拉或克拉克斯抛出片面线索。

### 5.2 天庭机械寺

开局应围绕“机械筑基试炼与小七问题”展开：

- 接受器灵长老的本命法器适配。
- 询问冷凝御剑考核标准。
- 和小七谈 heart.exe。
- 前往演武场或藏经阁准备试炼。

失败推进：

- 考核评分下降，但暴露神经接口异常。
- 小七被安全协议盯上。
- 冷凝提出更严苛但具体的训练条件。

### 5.3 第7区避难所

开局应围绕“地表人是否值得信任”展开：

- 向老王说明地表现状。
- 接受苏珊基础体检。
- 听阿杰展示全息投影。
- 查看旧商场地图和补给路线。

失败推进：

- 委员会信任下降，但给出试探任务。
- 物资时钟推进，配给更紧。
- 苏珊发现变异线索但引发伦理冲突。

## 6. 验收

- 新建任意预设世界后，输入区 chips 能给出 3-4 个剧本相关建议。
- 右侧局势“可选行动”不再只显示泛泛的“推进任务”。
- Prompt 中包含剧本流程指南。
- 玩家输入“我该做什么”时不会推进剧情，而是获得当前流程提示。
- 不泄露 NPC 未解锁秘密。
- 失败、部分成功、拖延均能推动时钟、关系、线索或位置变化。
